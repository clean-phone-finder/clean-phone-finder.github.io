Мета сторінки — допомогти користувачу сформувати якісний текстовий prompt для вибору смартфона для повсякденного використання без нав’язливої реклами, spyware та adware, які можуть агресивно встановлювати застосунки без можливості це зупинити, а також з урахуванням реальних hardware проблем, таких як перегрівання, тротлінг, проблеми з акумулятором і стабільністю роботи компонентів, і software проблем, зокрема зависань, крашів, нестабільних оновлень, багів прошивки та деградації продуктивності з часом, на основі реальних користувацьких відгуків і обговорень.

Сторінка має бути повністю статичною, responsive, з мінімалістичним CSS, підтримувати світлу і темну тему з можливістю перемикання. Увесь текст інтерфейсу, плейсхолдери, SEO та згенерований prompt мають бути англійською мовою.

Форма має містити необов’язкове поле для введення назви смартфона з datalist популярних моделей, який буде заповнений пізніше. Має бути необов’язкове поле для вибору країни з datalist країн, який також буде заповнений пізніше. Має бути необов’язкове поле для вибору мови відповіді з datalist мов, який буде заповнений пізніше. Усі datalist будуть динамічно заповнюватися зі списків у JavaScript, наче налаштування. Усі поля можуть бути порожніми або заповненими незалежно одне від одного.

Має бути textarea у режимі лише для читання, з якого можна скопіювати згенерований результат, а також окрема кнопка Copy. Значення textarea має автоматично оновлюватися при зміні назви смартфона, країни або мови.

Логіка формування тексту в textarea така. Якщо назва смартфона, країна і мова відсутні, формується загальний англомовний prompt для worldwide вибору смартфона на основі реальних користувацьких обговорень і відгуків, з обовʼязковим використанням Reddit як одного з основних джерел. У цьому prompt має бути явна вимога попереджати користувача, якщо моделі менше одного року і відповідно недостатньо незалежних відгуків, а також пропонувати альтернативні моделі без описаних проблем.

Якщо введена лише країна, prompt має враховувати регіональні особливості цієї країни, доступність моделей, регіональні прошивки, обмеження та специфічні скарги користувачів з цього регіону, при цьому Reddit залишається обовʼязковим джерелом.

Якщо введена лише назва смартфона, prompt має бути сфокусований на аналізі цієї конкретної моделі з перевіркою hardware і software проблем, скарг користувачів на рекламу, spyware, adware, примусові встановлення застосунків, регіональні відмінності прошивок і час присутності моделі на ринку.

Якщо введені назва смартфона і країна, prompt має одночасно враховувати обидва параметри без ігнорування жодного з них.

Якщо введена мова, незалежно від інших полів, у prompt має бути чітка вимога, щоб ШІ-асистент надавав відповідь саме цією мовою. Якщо мова не вказана, відповідь має бути англійською.

У кожному варіанті сформований prompt, який генерує ця HTML-сторінка, має жорстко забороняти ШІ вигадувати факти, робити припущення без джерел або подавати неперевірену інформацію. Формулювання мають вимагати опори лише на реальні користувацькі звіти, обговорення і публічні джерела з явною вказівкою, що Reddit є обовʼязковим, але не єдиним джерелом. Додатково prompt, згенерований цією HTML-сторінкою, має бути суцільним текстом без використання emoji, без HTML, без Markdown і без будь-якого спеціалізованого форматування.
