# Prompt generator: Find smartphone without annoying features

A static web tool that generates AI prompts to help you find smartphones without intrusive ads, spyware, or hardware issues based on real user reviews.

## 🌐 Live Demo

Visit: [https://clean-phone-finder.github.io/](https://clean-phone-finder.github.io/)

## ✨ Features

- **Smart Prompt Generation**: Creates detailed prompts based on your search criteria
- **Multiple Search Modes**: Search globally, by country, by smartphone model, or combine parameters
- **Real User Reviews**: Prompts emphasize Reddit and other community discussions as primary sources
- **Hardware & Software Focus**: Covers overheating, throttling, battery issues, crashes, and more
- **Privacy-Focused**: Identifies models with intrusive ads, spyware, or forced app installations
- **Multi-language Support**: Generate prompts for AI responses in various languages
- **Dark/Light Theme**: Comfortable viewing in any lighting condition
- **One-Click Copy**: Instantly copy generated prompts to clipboard

## 🚀 Usage

1. Optionally enter a smartphone name (e.g., "Samsung Galaxy S24")
2. Optionally select a country for regional-specific analysis
3. Optionally choose a response language
4. Copy the generated prompt
5. Paste it into any AI assistant (ChatGPT, Gemini, Copilot, Claude)

## 🎯 What Makes This Tool Special

- Requires AI to cite **real user experiences** from Reddit and tech forums
- Warns about models less than 1 year old that lack sufficient reviews
- Focuses on **long-term reliability** and hidden issues
- Considers **regional firmware differences** and country-specific problems
- Explicitly forbids AI from inventing facts or making assumptions

## 🛠️ Technical Details

- Pure HTML, CSS, and JavaScript
- No frameworks or external dependencies
- Fully responsive design
- Client-side only (no data collection)
- Theme preference saved in localStorage
